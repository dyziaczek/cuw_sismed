\# CUW\_SISMED



Centrum Umawiania Wizyt lekarskich dla przychodni medycznej SISMED.



\## Funkcje systemu



> rezerwacja wizyt

> anulowanie wizyt

> zamiana wizyt
> system ostrzeżeń

> blokada usług (po 3 ostrzeżeniach)

> kalendarz lekarzy (z filtrem)



\## Godziny pracy 



Przychodnia pracuje od poniedziałku do soboty w godzinach 7:00 - 18:00



\## Wykorzystane technologie



> Windows application

> Git + GitHUB

> SQLite (baza danych)
